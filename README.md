## Запуск
Реализован простейший CLI интерфейс. Чтобы его запустить, в корневой папке проекта прописать команду:
```bash
python -m src.main
```

---

## Синтаксис для CLI
### Список доступных команд
```bash
help
```

### Выйти из CLI
```bash
exit
```

### Начать симуляцию
```bash
run <steps> <seed>
# Пример 1: run 20 2 (псевдорандом)
# Пример 2: run 20 None (рандом)
```

## События в казино
Каждый шаг симуляции запускает одно из 6 случайных событий:
```bash
"ГУСЬ АТАКУЕТ" - WarGoose атакует случайного игрока и отнимает деньги
"ОРУЩИЙ КУСЬ" - HonkGoose криком изменяет баланс игрока (может помочь или навредить)
"Ну эта ставка точно зайдет" - Игрок делает ставку на рулетке (выигрыш/проигрыш)
"Робин Гусь" - Гусь забирает фишки у богатого игрока и отдает бедному
"НЕ НАДО ПАНИКИ" - Гусь пугает игрока, заставляя обнулить баланс или упасть со стула
"Удвой или обанкроться" - Рулетка: четное число удваивает бюджет, нечетное - банкрот
```

---

## Особенности
- Игроки могут конвертировать фишки в деньги при недостатке средств
- Банкроты автоматически исключаются из игры
- Поддержка seed для воспроизводимости результатов
- Детальное логгирование всех событий
